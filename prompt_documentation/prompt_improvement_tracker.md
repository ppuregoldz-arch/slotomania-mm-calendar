## Scoring Guide
- **C**: Clarity (1-5)
- **S**: Scope (1-5)
- **C**: Complexity (1-5)
- **A**: Actionability (1-5)

## Improvement Categories
- **Metrics**: Added specific metrics
- **Timeframe**: Added timeframe specification
- **Segments**: Added segment definition
- **Objective**: Added business objective
- **Currency**: Clarified currency separation
- **Aggregation**: Specified aggregation methodology

## SM-Specific Patterns
Track common improvement patterns:
- Revenue analysis prompts
- User engagement prompts
- Economy analysis prompts
- Product performance prompts
- Churn analysis prompts

This tracker helps monitor prompt quality improvements over time for SM data analysis.
