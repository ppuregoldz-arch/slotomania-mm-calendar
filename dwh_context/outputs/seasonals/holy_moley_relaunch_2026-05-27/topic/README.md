# Holy Moley Feature Re-launch - May 2026

## Project Context

**Business Question**: Re-launch the Holy Moley seasonal feature after ~2 years of inactivity to provide players with a familiar pick-and-reveal gameplay experience similar to Blast.

## Feature Background

**Holy Moley** is a returning seasonal feature where players chase a mole hiding in SlotoCity's underground subway system. This feature is part of the **Seasonals squad portfolio**. Players reveal doors to win prizes.

**Note**: Detailed mechanics and prize structures require investigation through historical data analysis.

### Key Feature Characteristics
- **Feature Type**: Seasonal feature with door-reveal mechanics
- **Previous Activity**: Launched in past years, inactive for approximately 2 years
- **Theme**: Underground subway/tunnel setting with mole character
- **Mechanics**: Door-reveal gameplay (specific mechanics require data discovery)

## Re-launch Context

This is not a new feature development but rather a **re-activation** of an existing feature that:
- Has established gameplay mechanics (to be analyzed)
- Has historical performance data (availability to be confirmed)
- Requires investigation of previous configurations and performance
- Needs validation of technical implementation and data structures

## Analysis Requirements

### Primary Focus Areas
1. **Historical Performance Review**: Analyze pre-discontinuation metrics
2. **Blast Comparison**: Compare mechanics and performance with current Blast feature

- Completion rates and player progression
