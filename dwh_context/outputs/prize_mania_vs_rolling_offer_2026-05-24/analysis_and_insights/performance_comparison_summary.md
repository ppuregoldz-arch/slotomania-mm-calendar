## Performance Analysis

### Revenue Performance
- **Rolling Offer Total Revenue**: $1.25M (5.1x larger than Prize Mania)
- **Prize Mania Peak Day**: $130,772 (May 22) - Strong single-day performance
- **Daily Revenue Comparison**: Rolling Offer averages $44,523/day vs Prize Mania's $121,440 on active days

### User Engagement
- **Rolling Offer Purchasers**: 31,150 unique purchasers across 28 days
- **Prize Mania Purchasers**: 23,157 purchasers across 3 active days
- **Purchase Penetration**: Prize Mania achieves ~11,500 daily purchasers when active

### Transaction Patterns
- **Rolling Offer**: 143,140 transactions (5,112/day average)
- **Prize Mania**: 25,922 transactions on active days (~8,640/day when active)
- **Average Transaction Value**: Similar ($8.71 RO vs $9.37 PM)

## Key Insights

### 1. **Feature Activity Patterns**
- **Prize Mania**: High-intensity, limited-time events (3 days active out of 28)
- **Rolling Offer**: Consistent daily availability and usage

### 2. **Revenue Generation Models**
- **Prize Mania**: Event-driven revenue spikes with concentrated user activity
- **Rolling Offer**: Steady baseline revenue stream with broader user reach

### 3. **User Engagement Quality**
- **Similar Average Spend**: Both features maintain ~$8.50-9.50 per transaction
- **Different Reach**: Rolling Offer engages 34% more unique purchasers
- **Intensity vs. Consistency**: Prize Mania shows higher daily peaks, Rolling Offer provides stable volume

## Business Implications

### Revenue Strategy
- **Prize Mania** functions as effective **revenue acceleration** during active periods
- **Rolling Offer** provides reliable **baseline monetization** infrastructure
- Combined approach maximizes both **peak performance** and **consistent revenue**

### User Base Impact

- No significant cannibalization evident (different usage patterns)
