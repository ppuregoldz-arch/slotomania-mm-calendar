# Gameplay Squad - Queries & Dashboard Intelligence

## Summary
This file contains dashboard intelligence and analytical patterns for the Gameplay squad, focusing on core game mechanics, machine performance, and player progression analysis.

## Key Analytical Areas

### Machine Performance Analytics
- **RTP Analysis**: Tracking actual vs. theoretical return-to-player percentages
- **Volatility Metrics**: High/medium/low volatility machine performance
- **Popular Machine Tracking**: Most played machines by user segment
- **Jackpot Analytics**: Progressive jackpot hit frequency and amounts

### Player Progression Analysis  
- **Level Advancement Rates**: Time to level progression by user segment
- **XP Generation Patterns**: Experience points earned per session/spin
- **Feature Unlock Analytics**: When players reach key milestones (level 72 for Clans, etc.)
- **Achievement Completion**: Platform-specific achievement rates

### Platform Comparison
- **Cross-Platform Performance**: PC/Browser vs. Mobile gameplay patterns
- **Login Method Analysis**: Performance differences by authentication type
- **Achievement Platform Comparison**: Google Play Games vs. Apple Game Center engagement

### Gameplay Engagement Metrics
- **Session Length Analysis**: Average gameplay duration by machine type
- **Bet Progression**: How players adjust betting patterns over time
- **Machine Switching Patterns**: Player behavior across different machine types
- **Feature Utilization**: Bonus feature trigger rates and player engagement
