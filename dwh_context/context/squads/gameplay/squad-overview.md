# Gameplay Squad Overview

## Mission
The Gameplay squad focuses on core game mechanics, slot machines, and fundamental gameplay systems that drive the Slotomania experience.

## Key Sub-Features

### Core Systems
- **[Machine Mechanics](./machine-mechanics/business-context.md)**: Slot machine payouts, RNG systems, bet mechanics, jackpots
- **[Game Progression](./game-progression/business-context.md)**: Level up system, XP mechanics, unlock progression
- **[Achievement Systems](./achievement-systems/business-context.md)**: Google Play Games and Apple Game Center achievements

### Navigation
- Main Squad Overview: This file
- [General Gameplay Context](./general-gameplay.md): Core gameplay mechanics and systems
- [Gameplay Queries](./queries.md): Dashboard intelligence and analytical patterns

## Scope
This squad covers the fundamental gaming experience mechanics that are platform-agnostic and core to the slot machine gameplay, excluding monetization features (covered by Purchase Tools) and specific timed features (covered by other squads).

## Key Metrics
- RTP (Return to Player) percentages
- Machine volatility and payout patterns  
- Level progression rates
- Achievement completion rates
- Session length and gameplay patterns
