# General Gameplay Context

## Core Game Mechanics

### Slot Machine Systems
**Payout Structure**: All slot machines in Slotomania have payouts generally above 90% RTP and are based on RNG (Random Number Generator).

**Bet Formula**: `(total bet / coins played at) x symbol multiplier`

**Machine Characteristics**:
- Can have any number of reels (typically 3-5)
- Variable paylines and bet structures
- Built-in bonus features (Free Spins, Pick games, etc.)
- Progressive jackpot integration where applicable

### Level and Progression System
**Starting Balance**: All new players start with 15 million coins at level 1

**Experience Points (XP)**:
- Only earned through spinning
- Based on bet amount and gameplay activity
- Required for level advancement
- Unlock new machines and features at specific levels

**Level Benefits**:
- Access to new slot machines
- Higher betting limits on existing machines
- Feature unlocks (Clans at level 72, etc.)
- Increased daily bonus values

### Login Methods and Platforms

#### PC & Browser
- Access via slotomania.com
- Login options: Facebook, Email, Guest (added March 2026)
- Email option includes phone number login method
- No achievements or game center integration

#### Android Devices  
- Login options: Facebook, Google Play Games, Guest
- Google Play Games integration provides achievements and "Welcome Back" messages
- Achievement system fully supported

#### Apple Devices
- Login options: Facebook, Apple ID, Guest  
- Apple Game Center integration (automatic for all login methods)
- Achievement system with "Welcome Back" messages
- Players can sign out of Game Center in device settings

## Game Lobby Structure

### Top Section (Left to Right)
- **Player Profile**: Contains Stash (seasonal tokens), status benefits, profile picture
- **Player Balance**: Current coin balance display
- **Gift Box**: 100 gifts limit per category, 10-day expiration (except Mystery Gifts and SlotoClub Cashback)
- **Store Button**: Access to Coin Store
- **Gems Balance**: Premium currency display  
- **Level and XP**: Current level and experience progress

### Middle Section
- **Mission Center**: Daily and ongoing challenges
- **Machine List**: Scrollable list of available slot machines
- **Machine Playlists**: Curated collections (e.g., Mega Pods playlist)

### Bottom Section  
- **NewsFeed Megaphone**: Game announcements and news
- **Daily Dash Widget**: Challenge progress display
- **Slotocards Widget**: Album collection access
- **Smash It Widget**: Mini-game access
- **SlotoClans Widget**: Social feature access
- **SlotoQuest Widget**: Adventure mode access
- **Lobby Bonus**: Collectible bonuses

## Machine Categories

### Standard Machines
- Regular RTP above 90%
- Variable betting ranges based on player level
- Standard bonus features

### High Limit Machines
- Higher minimum bets (typically 1M+ coins)
- Higher maximum bets available
- Enhanced payout potential
- Often exclusive to higher-level players

### Royal Jackpot Machines
- Progressive jackpot pools
- Ante bet requirements (typically 20% additional)
- Community-shared jackpot pools
- Higher volatility gameplay

### Themed Collections
- **Inner Circle Machines**: Exclusive to VIP players
- **Permanent Featured**: Always available with special features
- **Seasonal Integration**: Machines that tie into current events

## Risk Areas
- **RNG Compliance**: Ensuring fair play and regulatory compliance
- **Payout Accuracy**: Maintaining advertised RTP percentages
- **Level Progression Balance**: Preventing too fast/slow advancement
- **Achievement Sync**: Cross-platform achievement consistency
- **Login Method Conflicts**: Account merging and data preservation

## Technical Architecture
- **Client-Server Validation**: All spins validated server-side
- **RNG Certification**: Third-party verified random number generation
- **Cross-Platform Sync**: Account data synchronized across devices
- **Achievement Integration**: Platform-specific achievement APIs
- **Progress Tracking**: Persistent level and XP tracking
