# Purchase Tools Squad - Overview

**Squad**: Purchase Tools
**Focus**: Monetization features, virtual currency systems, and purchase optimization tools

## Squad Structure

### Sub-Features
- **[MGAP](mgap/)** - Main monetization tool
- **[LBP](lbp/)** - Lotto Bonus Premium purchase optimization  
- **[Dice Deluxe](dice-deluxe/)** - Premium dice-based feature
- **[AVIATOR](aviator/)** - Purchase tool feature
- **[Counter PO](counter-po/)** - Counter purchase optimization feature
- **[Prize Mania](prize-mania/)** - Mission-based rewards system
- **[SlotoBucks](slotobucks/)** - Virtual currency system
- **[Gems](gems/)** - Premium currency system
- **[Offers](offers/)** - Rolling Offers, Sticky Bundle, Reveal Your Deal
- **Config Buckets** - Configuration management (see general-purchase-tools.md)

### Key Areas
- Purchase funnel optimization
- Virtual currency management and flow
- Dynamic pricing and offers
- User segmentation for monetization
- Purchase conversion analysis
- Revenue tracking and optimization
