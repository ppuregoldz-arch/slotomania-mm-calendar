# Currencies - Queries & Analytics

## Summary
This file contains analytical patterns and dashboard intelligence for in-game currency performance, conversion tracking, and economic balance analysis within the Purchase Tools ecosystem.

## Key Analytical Areas

### Currency Economics
- **Coin Flow Analysis**: Earning vs. spending patterns across all coin sources and sinks
- **Gem Conversion Rates**: Real money to gems purchase conversion and usage patterns
- **Slotobucks Performance**: New vs. legacy Slotobucks adoption and usage analysis
- **Currency Balance Distribution**: Player balance segmentation and spending behavior

### Monetization Analytics
- **Gem Revenue Attribution**: Direct revenue impact and cross-feature spending patterns
- **Coin Purchase Drivers**: What motivates coin purchases and optimal pricing analysis
- **Slotobucks ROI**: Cashback program effectiveness and retention impact
- **Cross-Currency Behavior**: How players use different currencies in combination

### Feature Integration Analysis
- **Currency-Gated Feature Usage**: Which gem-based features drive the most engagement
- **Machine-Specific Currency Performance**: Slotobucks-enabled machine success rates
- **Seasonal Currency Impact**: Hero coins and temporary currency effectiveness
- **Balance Maintenance Patterns**: How players manage multiple currency balances

### Economic Health Metrics
- **Inflation Tracking**: Currency value stability and earning rate analysis
- **Sink Effectiveness**: Are currency sinks properly balanced with sources
- **Player Progression Impact**: How currency availability affects game progression
- **Whale vs. Minnow Behavior**: Currency usage patterns across spending segments
