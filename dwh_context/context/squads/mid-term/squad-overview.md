# Mid-term Squad Overview

## Squad Mission
The Mid-term squad focuses on **medium-duration rotating features** (typically 1-4 weeks) that provide variety and engagement between major seasonal events. These features include progressive systems, collection mechanics, betting enhancements, and special event formats that maintain player interest and drive incremental monetization.

## Key Business Areas
- **Engagement Bridging**: Maintaining interest between major seasonal events
- **Content Variety**: Rotating different mechanics to prevent feature fatigue  
- **Incremental Monetization**: Additional revenue opportunities through mid-term features
- **Player Segmentation**: Different mid-term features appeal to different player types
- **Retention Enhancement**: Regular content refresh keeps the game feeling fresh
- **Cross-Feature Integration**: Mid-term features often enhance or interact with core systems

## Technical Framework
- **Rotation Management**: Systematic scheduling and deployment of mid-term features
- **Performance Tracking**: Analytics for engagement, retention, and revenue impact
- **A/B Testing**: Feature variants and optimization experiments
- **Integration Systems**: Seamless integration with core game systems and other features
