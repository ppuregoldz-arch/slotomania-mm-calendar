# Globez - Business Context

## Feature Overview
**Globez** is a mid-term collection feature where players collect snow globes from different global regions (USA, Australia, Japan, France sets) through designated game play, complete regional sets for coin grab rewards, and engage in themed collection mechanics with geographic variety.

## Core Mechanics

### Globe Collection System
- **Regional Sets**: Four distinct geographic regions (USA/AUS/JPN/FRA) each with unique globe collections
- **Designated Games**: Specific machines where globes can be collected through gameplay
- **Set-Based Progression**: Complete individual regional sets for rewards
- **Cross-Regional Collection**: Players can work on multiple regional sets simultaneously

### Reward System
- **Coin Grab Rewards**: Set completion unlocks coin grab bonus games
- **Regional Rewards**: Each geographic set offers themed rewards and bonuses
- **Progressive Rewards**: Increasing value rewards for completing multiple regional sets
- **Payout Distribution**: Comprehensive reward distribution across all participation dates

### Collection Mechanics
- **Spinning Integration**: Globes earned through regular gameplay on designated machines
- **Set Completion Tracking**: Progress monitoring for each regional collection
- **Collection Goals**: Multiple regional objectives providing varied engagement targets
- **Themed Content**: Geographic theming creates variety and collection appeal

## Business Goals

### Primary Objectives
- **Geographic Engagement**: Themed regional content appeals to diverse player interests
- **Collection Variety**: Multiple simultaneous collection goals extend engagement
- **Reward Motivation**: Coin grab rewards provide clear collection incentives

### Strategic Value
- **Content Differentiation**: Geographic theming distinguishes from other collection features
- **Multi-Track Engagement**: Multiple regional sets allow players to choose focus areas
- **Reward Excitement**: Coin grab mechanics add interactive reward collection
- **Cross-Feature Enhancement**: Collection activities enhance designated machine engagement
