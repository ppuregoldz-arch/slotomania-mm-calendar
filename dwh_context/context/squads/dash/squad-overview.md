# Dash Squad - Overview

**Squad Focus**: Daily engagement mechanics, user retention, and monetization through structured progression systems
**Primary Features**: Daily Dash, Daily Dash Max, Super Dash, challenge systems, progression tracking

## Squad Mission

The Dash squad encompasses all daily engagement mechanics that drive user retention, session frequency, and monetization through structured daily objectives, premium subscriptions, and progression systems.

## Sub-Features

### Core Daily Engagement Features
- **[Daily Dash](daily-dash/)** - Core daily challenge mechanics and standard progression
- **[Daily Dash Max](daily-dash-max/)** - Premium bi-weekly subscription with enhanced rewards and missions
- **[Challenge System](challenge-system/)** - Individual challenge mechanics and completion tracking

## Navigation Guide

For **Daily Dash** (core feature) analysis:
→ Use [daily-dash/](daily-dash/) folder for standard daily challenge mechanics

For **Daily Dash Max** premium subscription analysis:
→ Use [daily-dash-max/](daily-dash-max/) folder for bi-weekly premium progression

For **Challenge System** mechanics:
→ Use [challenge-system/](challenge-system/) folder for individual challenge tracking

For **squad-wide analysis**:
→ Use this folder's root files for cross-feature dash analytics
