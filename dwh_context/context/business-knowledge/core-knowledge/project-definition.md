# Project Objective: Product Usage Analysis in Slotomania

## Background

This project focuses on analyzing product usage patterns in the Slotomania game, aiming to understand usage trends and their implications on monetization metrics.

## Goals

1. **Analyze changes in product usage**: Examine how usage patterns have shifted over time across different products.

2. **Revenue metrics analysis**: Evaluate Revenue, ARPPU, and other payment-related KPIs to assess the impact of these usage changes.

3. **Time-based comparison**: Compare data from the last 30 days versus the last 90 days to identify short- and long-term trends.

4. **Value for Money**: Assess players' perceived value of products over different time periods.

## Methodology

- **Data collection**: Using Cursor and other analytics tools to gather the necessary data.

- **Data analysis**: Apply segmentation techniques and clustering to detect patterns and trends.

- **Insights presentation**: Summarize findings and provide actionable recommendations.

## Final Deliverable

A detailed report including charts, tables, and operational recommendations to improve understanding of product usage and monetization in the game.
