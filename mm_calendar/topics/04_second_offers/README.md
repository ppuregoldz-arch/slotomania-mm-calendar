# 04 — Second offers (הצעה שנייה / VFM)

## מה זה
מעבר ל-DD: **לפחות הצעה אחת** עם ערך מוניטיזציה (RYD / Rolling / Buy All / Decoy / Coin Sale row / MGAP לא מחליף).

**לא נספר כ-second offer:** Clan-Dash, Dash Pass (VFM rules).

## פלטפורמות

| הצעה | מבנה קצר | תמחור נפוץ |
|------|-----------|------------|
| **RYD** | Coins + Gems + hook + RDS + %SB; **תמיד פרס מעבר ל-SB** | High |
| **Buy All** | 2 denoms: Coins+RDS \| Gems+GGS + extras | M/H |
| **Decoy / Bonanza** | d1 Coins+RDS · d2 Gems+GGS (decoy) · **d3 = ALL d1+d2 prizes + 4 RDS + (1 or 2) GGS + hook d3-only** | M-H-H (d3 H/Max) |
| **Prize Mania** | חבילת פרסים (RDS, Wild, SB, boosters…) | High |
| **Coin Sale** | שורת חנות % — בדרך כלל עם sale שישי-שבת | High |
| **Coin Sale** | שורת חנות % — בדרך כלל עם sale שישי-שבת | High |
| **Rolling** | ראה `03_rolling_offer/` | M/High |
| **Stash Booster** | מכפיל קוינס על **DD + הצעה ראשית**; לא עם Rolling/MGAP | 2× CZ configs — ראה `../../stash_booster.md` |

## Stash Booster (HARD exclusions)
- **לא** ביום **Rolling** · **ביום Stash — Stash מחליף MGAP** (לא שורת MGAP + Stash) · **≤1/שבוע** · drain באותו יום + ימים אחריו.
- בורד: **RLAP** = Stash Booster task (post-FL).

## רוטציה (HARD)
{Buy All, Bonanza, Rolling, RYD} — לפזר; לא אותו משפחה יום-after-יום.

## מה אפשר לשבץ
- כל SKU מהמאגר ב-slots (קלפים לפי בנק, Gold ברכישה בלבד)
- **RYD:** אסור 155% SB
- **Decoy d3 (HARD):** **כל פרסי d1 + d2** ב-bundle + **4 RDS** + **1 או 2 GGS** + hook (Wild / Shiny / 5★ Gold…) — ראה `offer_construction.md` §Decoy. **לא HARD:** חובה 2 GGS ב-d3.
