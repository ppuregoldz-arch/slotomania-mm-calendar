# 02 — Daily Deal

## מה זה
Bundle יומי רכישה — **פלטפורמה**: Coins + Gems + SKU מרכזי(ים) + תמחור.

## מבנה
- Slot מרכזי מהמאגר: Hammers, Wild, SB Wheel, קלף (3★–5★ Reg/Ace/Gold), Picks, עונה…
- **Multiple** = רכישות חוזרות (ברירת מחדל)
- **Wild / Shiny Limited** = **Once** → חובה **DD Multiple נוסף** באותו יום (בלי Wild/Shiny)

## מה אפשר לשבץ (דוגמאות סוג)
| סוג | דוגמה שם |
|-----|----------|
| Bundle | `100% SB + 8 Hammers` |
| Wheel | `Hammer Wheel` / `SB Wheel up to 500%` |
| קלף | `4★ Reg + SB Wheel` · `5★ Gold` · `4★ Gold` |
| Once + חלופה | `Shiny Limited Once` + `100% SB + Hammers` Multiple |
| BOGO | `6 Hammers BOGO` (לעיתים ללא ג׳מים) |
| עונה | `Parasheep + AS` · `5 SnL Dice` |
| מסונן / חלונות | Finishers/Non-Finishers · Night plan 00:00–06:00 |

## כללים קריטיים
- תמחור DD **≠** תמחור second offer (כששניהם ממוחרים)
- Gold רק ברכישה · קלפים מבנק השבוע
- DD signals היסטוריים: SB Wheel/Wild Supreme נראים טוב יותר בחלק מהחתכים; **Clan Pack אסור** והיסטורי בלבד. בדוק confidence במסמך ה-variant.
