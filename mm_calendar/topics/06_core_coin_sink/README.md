# 06 — Core / MES / Clan — ניקוז קוינס (Coin Sink)

## מה זה
**Core = צ׳אלנג׳ שדוחף שחקנים לשחק במכונות ולבזבז קוינס.** מנוע **ניקוז קוינס** (לא ג׳מס).
כולל MES, Spin Zone, Win Master, PYP, Custom Pod, Ace Heist.

> ניקוז **ג׳מס** (Shiny Show / Winovate) → `topics/11_gem_sink_shiny_winovate/`.

## תפקיד בקלנדר
- **≥1 פרומו גיימפליי-קוינס ביום** (שני: Dash Pass מספיק כמשטח יומי)
- בלנס קוינס גבוה → Core/MES/Custom Pod כבד
- MES 72h סוף-אלבום → wager חזק בכל הטירים

## פרסים (חשוב)
- Reg / Ace / Shiny / Wild — **לא Gold**, **לא שום SB** (Gold + SB = הצעות רכישה בלבד)
- פרס תואם שלב אלבום (`nivi_collector_album_prizes.md`)

## מה אפשר לשבץ
- Core themed / generic / machine / segment (`core_mes_references.md`)
- MES purchase / min-bet / סוף-אלבום 72h
- Spin Zone (עם/בלי Hammers Wheel), Custom Pod. ה-`+8.9%` ההיסטורי הוא revenue correlation משפחתי, לא הוכחת wager סיבתית.

## צפיפות
- ≤1 Core משחקי בלוח Monday · Dash Day = לא לערום פרומו-רבניו נוסף
