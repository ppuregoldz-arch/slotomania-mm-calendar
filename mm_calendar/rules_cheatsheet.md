# MM Calendar - Cheat Sheet (כל הכללים במקום אחד)

> סיכום מהיר לאייג'נט. הפירוט המלא בקבצים הייעודיים. **HARD** = חובה · **SOFT** = יעד.

## מקור סמכותי
- **גיידליין חודשי מהכלכלה** (`monthly_guidelines/YYYY-MM.md`) גובר על הכל. לקרוא לפני בנייה.

## ⭐ עיקרון-על: אופרים = פלטפורמות, לא מתכונים
- **כל אופר/פרומו הוא מיכל עם מבנה slots + מאגר פרסים משותף.** לתוך כל slot אפשר להזריק **כל** SKU מהמאגר (coins/gems/GGS/RDS/%SB/Hammers/Picks/PAB/AS/Dice/Boosters/קלפים Reg-Ace-Gold-Wild-Shiny/Superboom/48h access...).
- אותו אופר (Rolling More for Less / Buy All / Decoy / DD / RYD / Prize Mania) יכול לשאת **תוכן שונה לגמרי** בכל יום — לפי מטרה, תמחור, בנק קלפים, סגמנט וגיוון מול שאר היום.
- ה"דפוס הנפוץ" הוא ברירת-מחדל בלבד, **לא נעילה**. בעת שיבוץ: בחר פלטפורמה → קבע מטרה → מלא כל slot מהמאגר. פירוט מלא ב-`offer_construction.md`.

## Always-On (HARD - כיסוי רציף)
- **Short Term** (Blast/Battlesheep/SNL) - תמיד ≥1 חי; עונות גב-אל-גב: **Blast/Battlesheep ~5 ימים** · **SNL 3–4 ימים**.
- **Mid Term**: Winovate (8-9 ימים) + Mega Pods (Mon-Mon) תמיד חיים; רוטציה Quest/Globez/Figz (אחד בכל פעם).
- **Album**: מחזור פעיל תמיד (~70 יום / 10 שבועות).
- **בכל יום להציג אילו עונות פתוחות.**

## חובות יומיות (HARD)
- **Daily Deal** ≥1 (Coins+Gems+SKU). ברירת מחדל **multiple** (רכישה חוזרת). **DD עם Wild או Shiny Limited = once-per-player** → חובה לשבץ **DD multiple נוסף** שמחליף אותו לאחר הרכישה (ללא Wild/Shiny; רצוי SB Wheel/Hammers).
- **הצעה שנייה** אמיתית מעבר ל-DD (Buy All/Bonanza/Rolling/RYD/Coin Sale/MGAP). טופרים לא נספרים.
- **שני תמחורים שונים ביום** (DD + הצעה שנייה, ברמות Pricing שונות).
- **ADS PO** יומי - **פרסים נמוכים בלבד** (Coins/Gems/Clan Pack/reg נמוך/season). אסור Gold/Shiny Limited/Ace גבוה/Wild.
- **Core** (coin sink) ≥1 - צ'אלנג' עם פרס בסוף.

## עוגנים שבועיים (HARD)
- **שני = Dash Day** (Dash Pass עונה חדשה + Monday Max Deal + Time Limited Prize + Clan Go) + Mega Pods renew.
- **ראשון = Dash Max**. **X2 Dash Points** ~2/שבוע.
- **רביעי = Piggy** ("break for <prize>" - חובה לציין פרס; רכישה -> מותר Gold).
- **Lotto Bonus peak**: שלישי->רביעי 00:00 + שבת->ראשון 00:00, **על ה-Night Plan**.
- **Shiny Show 3/שבוע** (וריאציה שונה; העדף Joker/All Cards/Wild Guaranteed, המעט Crazy with Aces).
- **סופ"ש = Sale — שישי + שבת בלבד (לא ראשון)**. **Black Diamond** (VIP) 1/שבוע. **Rolling More for Less** 1/שבוע מקס' + **cooldown ~1.5-2 שבועות** (לא פעמיים בשבוע — נשחק). **חמישי = Golden Spin**.
- **~פעם ב-1.5 שבועות**: Dice promo · Prize Mania · Mystery Buy All · Counter PO.
- **Spinner Clash**: פעם בשבועיים לכל היותר.
- **Dice**: מוצר חינמי (בונוס קוינס); "1" מטריג Deluxe (מכפילים גבוהים). 3 סוגי פרומו: Booster גיימפליי · הגדלת טריגר Deluxe · ערך על רכישת Deluxe.

## עוגנים חודשיים
- **Machine Launch ×2** (תאריכי מח' סלוטים) - Sneak Peek לפני; Slot Smashes/MES tie-in **רק ביום ההשקה**.
- **Price Cut 20% ×2**. **Counter PO ×2** (on top). **MGAP BOGO ב-1 לחודש** (HARD).

## Caps (HARD - מהגיידליין החודשי; דוגמת יולי)
- **Hammers**: **מקור יחיד ליום (HARD)** — רק מוצר אחד נותן Hammers. Rolling יכול לפרוס על פני denoms/סייקלים שלו, אך **לא במוצר נוסף** באותו יום; שאר ההצעות → פרס חלופי (Picks/PAB/קלף). ≤4/שבוע. Pricing->SKU: Med 3-6 / High ≤8 / Max ≤12.
- **פרס עונת Mid-Term**: **Globez / Figz → 3000 Hero Coins** (לא Booster) · Quest → Quest Booster.
- **SKU עונת Short-Term (חייב להתאים לעונה החיה)**: **Blast → Superboom/PAB** · **Battlesheep → Parasheep/Airstrike** · **SNL → Dice/Multiwheel/Shield**. אסור SKU של עונה שלא באוויר. ⚠️ **SnL Dice בכפולות של 2/3 (לא 1).**
- **MGAP**: **≤3 פרומואים/שבוע** (לפי גיידליין ניבי; יולי=3). **Matched לא עם sale**. מרווח **≥2 ימים** בין פרומואי MGAP. לא בימי שני. **אין MGAP+Bucks** (מגיב חלש — רק BOGO/Bigger/Matched).
- **MGAP BOGO**: חובה ב-1 לחודש · **אסור ביום האחרון** · **אסור ביום sale** · בשאיפה **בכל שבוע** · **עדיף יום אחרי sale** (למניעת hangover).
- **Gems**: x2 GGS ≤2/שבוע; Gems Sale ≤4/חודש.
- **Dice Booster 24h** ×3/שבוע standalone.

## הדרות הדדיות (HARD)
- **VFM גבוה על קוינס - עד אחד ביום**: {MGAP BOGO, **MGAP Matched**, Rolling More for Less (כולל Buy More for Less), Coin Sale, Extreme Stamp, Price Cut}. (חריג: אירוע-ענק.)
- **MGAP Matched** לא עם sale. **Limited PO** 100% sale בלבד, לא עם MGAP. **Extreme Stamp** לא עם Wild Card, לא ביום sale (חריג 4thJ), ב-rolling מהמחזור ה-2. **DD BOGO** -> task config.
- **אין x2 GGS עם Gems Sale** באותו יום.
- **אין לשבץ Clan Pack** (כפרס/הצעה/Shiny Show Clan Pack Guaranteed) - הוסר לחלוטין.
- **אין פרומואים חזקי-רבניו בימי שני** (MGAP/Coin Sale/Rolling/Prize Mania/offers גדולים) - Dash Day מניב מספיק.

## קלפים
- **ערך עולה לקראת סוף אלבום** (שבוע אחרון = קלפים גבוהים/הרבה).
- ⭐ **שבוע אחרון לאלבום (HARD)**: בהצעות הרכישתיות קלפי הפרס = **Wildים בלבד** (לא Reg/Ace/Gold) — רק Wildים מזיזים בתקופה זו.
- **Gold רק ברכישות** - אסור כפרס צ'אלנג' משחקי (Core/PYP/MES). פרס צ'אלנג' = Reg/Ace/Shiny/Wild.
- **Ace** רק מגיימפליי (לא נמכר). **Shiny Limited** רק מפרומו (~1/שבוע, ערוץ בתשלום).
- **Wild** ≤1 per source. חלוקת הקלפים = **מאגר/תקרה, לא חובה**.
- ⭐ **בנק הקלפים**: כל פרס קלף בכל פרומו (Shiny Show/Core/ADS/DD/Offers) חייב להיות **מתוך בנק הכלכלה של אותו שבוע**. אין לתת סוג שלא בבנק; כמות ≤ בנק; 1★/2★ on top.
- **Where to get cards** (in-app): 3 שבועות / שבועיים / שבוע לפני סגירה, ואז כל יום עד הסגירה.

## תזמון (HARD)
- **Promo Time = 11:00 UTC** · **Night Plan = 00:00 UTC**.
- כל **Time-Limited** עולה אחרי Promo Time (≥12:00). **Boosted Gemback = 5h** · **x2 GGS = 3h**.

## Prize Mania - תוכן
- עם Wild Gold: `4 RDS + Wild Gold + 2 GGS + Dice Booster + coins + gems` · בלי: `coins + gems + 4 RDS + 1 GGS + 6H + 100% SB`

## מבנה הצעות - חותמות ובסיסים
- **Buy All** = 2 denoms: **Coins+Red Stamps** · **Gems+Gold Stamps** (+ פרסים על שניהם).
- **Decoy/Bonanza** = 3 denoms: d1/d2 כל אחד או Coins+RDS או Gems+GGS (+ פרסים); d3 = **שילוב שניהם + לעיתים on-top**.
- **RYD** = Coins+Gems+hook+RDS+%SB — **תמיד פרסים נוספים מעבר ל-SB**; **אסור 155%** (תמתי Cinco de Mayo).
- ⚡ **ימי Extreme Stamp**: כל ההצעות מחליפות **RDS → Extreme Stamp**, ו**4 RDS → 2 Extreme Stamps** (ערך גבוה יותר).

## כלכלה / ניקוז בלנס
- **ניקוז קוינז**: פרומו גיימפליי (Core) בכל יום.
- **בסייל Gems/Coins + יום אחרי**: פרסי גיימפליי משודרגים (prize↑) לדחיפת בזבוז בלנס.
- **מגבירים רוחביים** (Gemback/GGS/Extreme Stamp/Jumbo/Happy Hour): KPI = **הכנסת כל ההצעות** (לא רבניו ישיר). לשבץ בחלונות הרמת מכירות. **Boosted Gemback מאושש בדאטה**: +13M/יום צריכת ג'מס בכלל המשחק + halo רחב (5/6 מוצרים עולים). ראו `boosted_gemback_impact.md`.

## Stash Booster (post-FL · HARD)
- **Triggers:** **Daily Deal** + **main offer** (RYD/Decoy recommended; Buy All / Mystery Buy All OK).
- **Forbidden same day:** **Rolling Offer**.
- **MGAP:** on Stash days **Stash replaces the MGAP calendar promo** (no separate MGAP row); MGAP on other eligible purchases unchanged in product.
- **Cadence:** ≤ **1/week** recommended · **Drain:** same day + following days need balance-draining promos.
- **Configs (HARD):** CZ **0–159.99** → UP TO **×24** Coins (full cycle) · CZ **160+** → UP TO **×36** Coins · **2 configs + 2 comms** always.
- **QA:** DD + main offer buttons show **Stash Booster**, not Extreme MGAP; other MGAP-eligible purchases unchanged.
- **Monday label:** **RLAP** rows = Stash Booster task · `stash_booster.md`.

## סדר בניית יום
1. הצעות (DD + Other offers) → 2. מחזקי רבניו (Amplifiers/toppers) → 3. Gameplay → 4. כל השאר → 5. **ADS** (אחרון).

- ⭐ **בסיס ליפט = ממוצע ≥20 יום אחרון (trailing 20d)** — לעולם לא להשוות ליום בודד (±יום). אין להשתמש בעמודת %Difference יום-מול-יום מדשבורדים.
